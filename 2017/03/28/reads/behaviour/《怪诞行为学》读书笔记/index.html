第一章：相对论的真相
案例：电子书：59元；纸质书：125元；纸质书+电子书：125元

我们总是喜欢攀比

第二章：供求关系的失衡

要人们渴望做一件事，只需要做这件事的机会难以获得即可

锚定价格对之后的心里估价的影响

你给我钱听我朗诵和我给你钱让你听我朗诵；然后免费朗诵；会分别带来愉悦和痛苦的感觉

第三章：免费的代价

买二送一的冰淇淋：我们真的需要两个么

免费的最大的问题在于：它引诱你在它和另一件商品之间挣扎，并引导我们做出不明智的决定

轮流做东比AA更能增加友谊

第四章：社会规范的成本

社会规范与市场规范：一旦社会规范与市场规范发生碰撞，社会规范就会退出。换言之，社会规范很难重建(罚款对迟到作用不大)

社会性质的鼓励与荣誉上的鼓励，很少为公司采用。但却是有效的能够激励我们的社会奖赏

奖品比奖金更容易培养员工的忠诚度

第五章：性兴奋的影响(不要在情绪极度亢奋时做决定)

我们每一个都包含一个黑暗的本我，一个自我和一个能不可预测的从超我那里篡权控制权的兽性

在情绪极度亢奋时，一下子或一念之差，一切都变了。

你会跟一个50岁的老女人做爱么？

第六章：拖沓的恶习与自我控制

习惯性拖沓的根治手段：给自己设置底线的机会(分别设置三次不同的截止日期)，选择自己喜欢的行动路径。这种做法可能不如强制性规定那样有效，但是他能够帮助自己推向正确的方向

第七章：所有权的个性

三大非理性怪癖：

1.对我们已经拥有的东西迷恋到不能自拔(卖主的估价总比买主高)

2.我们总是���注意力集中到我们会失去什么，而不会是得到什么

3.我们经常假定别人看待交易的角度和我们一样

宜家效应���我们对某种事物投入的劳动越多，对它的感情越深

虚拟所有权：我们在实际拥有所有权之前就对某物产生了拥有的感觉(拍卖)

第八章：多种选择的困境

第九章：预期的效应

直接了当的告诉人们某种东西可能味道不好，结果十有八九的人会跟着你走，他们之所以这样，不是因为体验，而是因为预期

不要低估展示的力量：盒饭放在泡沫盒里与称在盘子里的味道不同
