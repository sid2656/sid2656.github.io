考研目标分数：220分以上

    • 逻辑 60-10
    • 写作 65-15
  • 英语：50分
    • 阅读 50-20
    • 翻译 15-10
    • 写作 25-15

详细时间计划

1. 基础学习：

时间范围：【暑期】08月01日 - 09月01日

  • 工作日：
    • 5:30 - 7:00 : 背英语单词（结合记忆曲线）每日50个单词；
    • 7:00 - 7:50 : 早餐、财经内容听说学习
    • 7:50 - 8:50 : 地铁快速复习昨日背诵单词；
    • 9:00 - 9:30 : 学习英语语法
    • 12:00 - 13:30 : 午餐、游戏签到活动
    • 13:30 - 13:50 : 午睡休息
    • 18:00 - 19:00 : 晚餐、游戏签到活动
    • 19:00 - 20:00 : 地铁回家复习今日背诵单词
    • 20:00 - 22:00 : 陪娃看书、准备休息
  • 周末：
    • 5:30 - 7:00 : 背英语单词（结合记忆曲线）每日50个单词；
    • 7:00 - 7:50 : 财经内容听说学习
    • 7:50 - 12:50 : 陪娃游戏、看书、挑房子
    • 13:30 - 18:00 : 看房子、陪娃游玩
    • 19:00 - 21:00 : 陪娃看书、准备休息
  • 暑期目标：
    • 每日一小时进行课程基础补弱学习
    • 如果可以尽量每周有两日晚间保持2小时的课程学习
    • 整体目标：
      • 8月完成管综课程基础学习
      • 英语单词核心词第一遍背诵完成

2. 强化学习：

时间范围：09月01日 - 11月01日

  • 工作日：
    • 5:30 - 7:00 : 背英语单词（结合记忆曲线）每日50个单词；
    • 7:00 - 7:50 : 早餐、财经内容听说学习
    • 7:50 - 8:50 : 地铁快速复习昨日背诵单词；
    • 9:00 - 9:30 : 学习英语语法
    • 12:00 - 13:30 : 午餐、游戏签到活动
    • 13:30 - 13:50 : 午睡休息
    • 18:00 - 19:00 : 晚餐、游戏签到活动
    • 19:00 - 21:00 : 基础补弱完成后，进行强化学习
    • 21:00 - 22:00 : 地铁回家复习今日背诵单词
  • 周末：
    • 5:30 - 7:00 : 背英语单词（结合记忆曲线）每日50个单词；
    • 7:00 - 7:50 : 早餐、财经内容听说学习
    • 7:50 - 12:00 : 个人成长视频制作（资料采集、编写、视频录制、后期、发布等工作）
    • 12:00 - 13:30 : 午饭、休息
    • 13:30 - 18:00 : 复习本周考研课程学习内容
    • 19:00 - 22:00 : 晚餐、游戏时间
  • 阶段目标：
    • 每日2-3小时进行课程强化学习
    • 整体目标：
      • 强化月完成管综课程重点题型强化
      • 管综写作完成5篇以上的范文并进行过默写尝试
      • 英语单词核心词第2、3遍背诵完成；非核心词第一遍学习；
      • 英语写作完成5篇以上的范文背诵并默写

3. 冲刺学习：

时间范围：11月01日 - 考试

  • 工作日：
    • 5:30 - 7:00 : 刷词、尝试阅读部分英文期刊；
    • 7:00 - 7:50 : 早餐、财经内容听说学习
    • 7:50 - 8:50 : 地铁快速复习背诵单词，200词；
    • 9:00 - 9:30 : 提分技巧学习，英语语法
    • 12:00 - 13:30 : 午餐、游戏签到活动
    • 13:30 - 13:50 : 午睡休息
    • 18:00 - 19:00 : 晚餐、游戏签到活动
    • 19:00 - 21:00 : 刷题，模拟
    • 21:00 - 22:00 : 地铁快速复习背诵单词，200词；
  • 周末：
    • 5:30 - 7:00 : 刷词、尝试阅读部分英文期刊；
    • 7:00 - 7:50 : 早餐、财经内容听说学习
    • 7:50 - 12:00 : 个人成长视频制作（资料采集、编写、视频录制、后期、发布等工作）
    • 12:00 - 13:30 : 午饭、休息
    • 13:30 - 18:00 : 针对薄弱的题型，专项主攻，进行强化学习
    • 19:00 - 22:00 : 晚餐、游戏时间
  • 阶段目标：
    • 每日2-3小时进行针对学习
    • 整体目标：
      • 冲刺月完成管综课程提分项的突破
      • 除真题考试外，额外每周进行一次模拟考试
      • 英语单词：核心词4、5遍；非核心词2、3遍
      • 英语写作：每周完成2-3篇文章的默写
